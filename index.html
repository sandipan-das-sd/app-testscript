Gyanoda App Comprehensive Test Script

1. Launch and Onboarding

1.1 App Installation and Launch

  1. Download the Gyanoda app from the app store
  2. Install the app on the test device
  3. Tap the app icon to launch
  4. Verify that the app opens without crashing

1.2 Onboarding Screens

  1. Check that onboarding screens appear
  2. Swipe through all screens, verifying content and images
  3. Ensure that the "Next" button works on each screen
  4. On the last screen, tap the "Getting Started" button
  5. Verify navigation to the login/signup screen

2. Sign Up Process

2.1 Sign Up Screen

  1. Tap "Sign Up" on the login screen
  2. Verify all required fields are present (name, email, password, phone, location)
  3. Test each field with valid and invalid data:
    • Name: Try empty, numbers, special characters
    • Email: Try invalid formats, existing email
    • Password: Test against password requirements
    • Phone: Try invalid formats, existing numbers
    • Location: Test auto-complete functionality
  4. Check that the "Sign Up" button is only active when all fields are valid
  5. Verify the "Terms & Conditions" checkbox functionality
  6. Test social sign-up options if available (Google, Facebook)

2.2 OTP Verification

  1. After successful sign-up, verify redirection to OTP verification screen
  2. Check that OTP is received via email/SMS
  3. Enter correct OTP and verify account creation
  4. Test incorrect OTP entry
  5. Verify "Resend Code" functionality (should be available after 60 seconds)
  6. Test OTP expiration (if applicable)

3. Login Process

3.1 Login Screen

  1. Verify all elements are present (email field, password field, login button, forgot password link)
  2. Test login with valid credentials
  3. Test login with invalid email format
  4. Test login with incorrect password
  5. Verify error messages for invalid login attempts
  6. Test "Remember Me" functionality if present
  7. Verify "Forgot Password" link functionality
  8. Test social login options if available

3.2 Forgot Password

  1. Tap "Forgot Password" link
  2. Enter email address and submit
  3. Verify reset instructions are sent
  4. Follow reset process and verify ability to set new password
  5. Test login with new password

4. Home Screen

4.1 Home Screen Layout

  1. Verify all components are present (header, search bar, course categories, featured courses)
  2. Check that user information is correctly displayed
  3. Verify that the search bar is functional
  4. Test the course category horizontal scroll
  5. Verify that featured courses are displayed correctly

4.2 Home Screen Functionality

  1. Test pull-to-refresh functionality
  2. Tap on a course category and verify filtering
  3. Tap on a featured course and verify navigation to course details
  4. Test navigation to other main sections (e.g., profile, notifications)

14. Final Review

14.1 User Experience Assessment

  1. Evaluate overall user experience and interface consistency
  2. Note any confusing or unintuitive elements

14.2 Bug and Improvement Documentation

  1. Compile a list of all observed bugs and issues
  2. Note suggestions for improvements or additional features
  3. Capture screenshots of any visual issues

14.3 Report Compilation

  1. Organize all findings into a comprehensive report
  2. Include steps to reproduce for any bugs found
  3. Prioritize issues based on severity and impact on user experience
